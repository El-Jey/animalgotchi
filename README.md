## Components
- `NodeJS >= v10.19.0`
- `npm >= v6.14.4`
- `Composer >= v1.10.1`

## Setup
- `git clone https://github.com/El-Jey/animalgotchi.git`
- `cd animalgotchi`
- `cp .env.example .env`
- Change .env config
     ### development
     - `make && make dev`

     ### production
     - `make && make prod`
- import **animalgotchi.sql** to database
- go to <http://localhost>
